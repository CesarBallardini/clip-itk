$FUNCNAME$
	SHA1()
$CATEGORY$
  Crypto
$SUMMARY$
	SHA-1 (Secure Hash Algorithm) is a cryptographic hash function with a 160
	bits output
$SYNTAX$
	SHA1(<cData>) --> cSHA1Digest
$ARGUMENTS$
	<cData> - string for decryption.
$RETURNS$
	SHA1 cData digest 20 bytes of output
$DESCRIPTION$
	A cryptographic hash function is a hash function, that is, an algorithm that
	takes an arbitrary block of data and returns a fixed-size bit string
$EXAMPLES$
	SHA1( "Hello" )		// Result: f7ff9e8b7bb2e09b70935a5d785e0cc5d9d0abf0
